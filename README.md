# WPP Close
A simple WIP whats app clone built with react js / nextjs / firebase


![CI Status](https://github.com/dennis90/wpp_clone/workflows/CI/badge.svg)

## Dependencies

* [Node.js 10+](https://nodejs.org/en/)

## Installation

```
npm install
```

## Usage

```
npm run dev
```


## Testing
```
npm test
npm run lint
```
